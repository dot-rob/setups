# Nord companion themes for XFCE/Debian

These files are Nord-styled companion color schemes for a consistent XFCE/Debian desktop.
They match the Nord palette used in the Geany, XFCE4 Terminal, Mousepad/GtkSourceView, and Rofi files.

## Files and install paths

- `nord.conf` → `~/.config/geany/colorschemes/nord.conf`
- `nord.theme` → `~/.local/share/xfce4/terminal/colorschemes/nord.theme`
- `nord.xml` → `~/.local/share/gtksourceview-4/styles/nord.xml`
  - alternate: `~/.local/share/gtksourceview-3.0/styles/nord.xml`
- `nord.rasi` → `~/.config/rofi/themes/nord.rasi`
- `nord.json` → `~/.config/tilix/schemes/nord.json`

## Install

Run:

```bash
./install.sh
```

Then select the schemes inside Geany, XFCE4 Terminal, Mousepad, Rofi, and Tilix as appropriate.
