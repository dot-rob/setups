# Qogir companion themes for XFCE/Debian

These files are Qogir-styled companion color schemes for the Qogir GTK/icon look.
They were adapted from the Nord files in this conversation and use a Qogir/Arc-style dark palette with a blue accent.

## Files and install paths

- `qogir.conf` → `~/.config/geany/colorschemes/qogir.conf`
- `qogir.theme` → `~/.local/share/xfce4/terminal/colorschemes/qogir.theme`
- `qogir.xml` → `~/.local/share/gtksourceview-4/styles/qogir.xml`
  - alternate: `~/.local/share/gtksourceview-3.0/styles/qogir.xml`
- `qogir.rasi` → `~/.config/rofi/themes/qogir.rasi`

## Install

Run:

```bash
./install.sh
```

Then select the schemes inside Geany, XFCE4 Terminal, Mousepad, and Rofi as appropriate.
